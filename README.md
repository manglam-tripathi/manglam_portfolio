# Manglam Portfolio

Personal portfolio website built with a **React + Vite** frontend and a **FastAPI** backend.

## Tech Stack

- **Frontend:** React, Vite, Tailwind CSS
- **Backend:** Python, FastAPI, Uvicorn

## Getting Started

### Install dependencies

```bash
npm run install:all
pip install -r backend/requirements.txt
```

### Run in development

```bash
npm run dev
```

Starts both frontend (Vite) and backend (FastAPI) concurrently.

- Frontend: `http://localhost:5173`
- Backend: `http://localhost:8000`

## Project Structure

```
manglam_portfolio/
├── frontend/   # React + Vite app
└── backend/    # FastAPI app
```
